Dataset Title:
Goldstonian Concordance Bible Knowledge Graph Dataset

Author:
Justin Goldston, PhD

Description:

The Goldstonian Concordance Bible dataset provides structured mappings between biblical passages, thematic classifications, cross references, and commentary nodes.

The dataset represents a prototype knowledge graph designed to support research in digital humanities and biblical studies.

It includes mappings across an expanded biblical canon and supports computational analysis of scripture relationships.
